## Design System Architecture
```
Primitives (Tokens)
├── Colors (semantic naming)
├── Typography (scale)
├── Spacing (4px/8px grid)
├── Shadows, Border radius
└── Animation curves

Components
├── Atoms (Button, Input, Badge)
├── Molecules (Form Field, Card, Toast)
├── Organisms (Header, Sidebar, Modal)
└── Templates (Dashboard, Settings)
```

## Design Principles
- Hierarchy: Size, weight, color, space, position
- Layout: 8px grid, 12-column, whitespace as feature
- Typography: 2-3 fonts, 1.5 line-height body, 65-75 char width
- Color: 4.5:1 contrast minimum, color blindness consideration
